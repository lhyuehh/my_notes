
# 期货数据说明与可视化解读

## 数据概览

我们使用的是**2022-2023年国内期货1分钟主连数据**，包含32个目标品种。

### 数据时间范围
- 开始：2022-01-04 09:00:00
- 结束：2023-12-29 14:59:00
- 频率：1分钟/条

---

## 数据字段说明

每个品种的数据包含以下32个字段：

### 基础信息字段
| 字段名 | 说明 |
|--------|------|
| **tradeDate** | 交易日期 |
| **dataDate** | 数据日期 |
| **barTime** | bar时间（如 09:00） |
| **ticker** | 具体合约代码（如 au2206） |
| **vt_symbol** | 完整合约标识 |

### 价格字段（不复权）
| 字段名 | 说明 |
|--------|------|
| **openPrice** | 开盘价 |
| **highPrice** | 最高价 |
| **lowPrice** | 最低价 |
| **closePrice** | 收盘价 |

### 价格字段（后复权 - hfq_前缀）
| 字段名 | 说明 |
|--------|------|
| **hfq_openPrice** | 后复权开盘价（用于回测） |
| **hfq_highPrice** | 后复权最高价 |
| **hfq_lowPrice** | 后复权最低价 |
| **hfq_closePrice** | 后复权收盘价 |
| **hfq_twap** | 后复权时间加权平均价（用于开平仓） |
| **hfq_vwap** | 后复权成交量加权平均价 |

### 成交量与持仓量
| 字段名 | 说明 |
|--------|------|
| **turnoverVol** | 成交量（合约数量） |
| **turnoverValue** | 成交金额 |
| **openInterest** | 持仓量（未平仓合约总数） |

### 平均价格
| 字段名 | 说明 |
|--------|------|
| **twap** | 时间加权平均价格（Time-Weighted Average Price） |
| **vwap** | 成交量加权平均价格（Volume-Weighted Average Price） |

### 其他字段
| 字段名 | 说明 |
|--------|------|
| **adj_factor** | 复权因子 |
| **limitUpPrice** | 涨停价 |
| **limitDownPrice** | 跌停价 |
| **preClosePrice** | 昨收价 |
| **CHG** | 涨跌额 |
| **CHG_pct** | 涨跌幅 |
| **contMultNum** | 合约乘数 |
| **minChgPriceNum** | 最小变动价位 |
| **tradeMarginRatio** | 交易保证金比例 |
| **tradeCommiNum** | 交易手续费 |

## 可视化图表解读

### 1. 价格和成交量图（{symbol}_price_volume.png）

**上图 - 价格走势**
- **蓝色线**：后复权收盘价（hfq_closePrice）
- **橙色线**：后复权TWAP（hfq_twap）
- **观察要点**：
  - 价格的整体趋势（上涨/下跌/震荡）
  - 价格波动的剧烈程度
  - TWAP与收盘价的关系（TWAP更平滑）

**下图 - 成交量**
- **柱状图**：每分钟的成交量
- **观察要点**：
  - 成交量的高峰期（通常开盘和收盘时段）
  - 价格大幅变动时是否伴随放量
  - 成交量与价格走势的关系

### 2. K线图（{symbol}_ohlc.png）

**简化版K线图**
- **黑色竖线**：影线（从最低价到最高价）
- **红色实体**：阳线（收盘价 ≥ 开盘价，上涨）
- **绿色实体**：阴线（收盘价 &lt; 开盘价，下跌）
- **观察要点**：
  - 价格的短期波动
  - 趋势的连续性
  - 关键支撑和阻力位

### 3. 持仓量图（{symbol}_open_interest.png）

**持仓量走势**
- **紫色线**：持仓量随时间的变化
- **观察要点**：
  - 持仓量增长：新资金进场，趋势可能加强
  - 持仓量减少：资金离场，趋势可能反转
  - 持仓量与价格的配合关系

### 4. 多品种对比图（multiple_symbols_comparison.png）

**标准化价格对比**
- 所有品种价格都标准化为从1开始
- **观察要点**：
  - 不同品种的相对强弱
  - 品种之间的相关性
  - 哪些品种趋势更强

## 为什么使用后复权数据？

在期货回测中，**必须使用后复权数据**，原因如下：

1. **合约换月**：期货合约有到期日，需要换月
2. **价格跳空**：换月时新旧合约价格可能不同
3. **连续性**：后复权消除了换月带来的价格跳空
4. **真实收益**：只有后复权数据才能计算真实的策略收益

**hfq_twap的重要性**：
- 任务要求：开平仓使用下一根bar的hfq_twap
- TWAP是时间加权平均价，更能代表实际成交价格
- 避免了使用收盘价可能带来的滑点问题

## 如何理解这个任务？

### 任务目标
构建一个**时序策略**，即每个品种独立交易，不进行跨品种套利。

### 策略逻辑
1. **分析价格走势**：通过均线等技术指标判断趋势
2. **生成交易信号**：金叉开多，死叉开空
3. **模拟交易执行**：
   - 当前bar结束时决定交易
   - 下一根bar的hfq_twap作为成交价
   - 不使用未来数据
4. **计算盈亏**：记录每笔交易的盈亏
5. **计算净值**：从1开始，每天一个点

### 回测评估指标
- **夏普比率**：风险调整后收益，越高越好
- **年化收益率**：策略的年化收益
- **最大回撤**：最坏情况下的亏损，越小越好
- **胜率**：盈利交易占比
- **盈亏比**：平均盈利/平均亏损
